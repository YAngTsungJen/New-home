# 主要功能
## 消費者：
1.可以瀏覽最新建案。<br>

2.點選查看更多查看單一建案詳細資料。<br>

3.點選加到預約看屋可將建案加入預約專區內。<br>

5.點選預約專區可瀏覽預約的建案。<br>

6.進入預約專區內的預約資料，查看預約資訊。<br>

7.填寫預約專區內的客戶資料進行預約，並送出表單。<br>

8.送出表單後連結到明細頁面，確認完成訂單。<br>

## 管理者：
1.填寫E-mail、Password登入管理頁面<br>

2.瀏覽全部建案清單<br>

3.新增新建案<br>

4.刪除單一建案<br>

5.編輯建案資訊<br>

6.瀏覽全部訂單資訊<br>

7.瀏覽全部上傳圖片<br>

8.刪除圖片<br>
